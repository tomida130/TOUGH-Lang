# TOUGH 言語 🤼

猿渡哲也の漫画『高校鉄拳伝タフ』や、その関連作品に登場する語録をもとにしたプログラミング言語。
LLVM コンパイラによってネイティブコードとして実行されます。

## 使い方

### 1. セットアップ

まず依存ライブラリをインストールします。

```bash
pip install llvmlite
```

### 2. コマンドの準備

以下のいずれかの方法で `tough` コマンドを使えるようにします。

**方法A: 環境変数に追加（推奨）**
フォルダ内の `add_to_path.bat` を実行し、PCを再起動（またはコマンドプロンプトを再起動）してください。

**方法B: ファイルの関連付け**
フォルダ内の `install_assoc.bat` を**管理者権限**で実行してください。`.tough` ファイルをダブルクリックで実行できるようになります。

### 3. 実行

```bash
# ファイルを実行
tough examples\hello.tough

# もしコマンドが使えない場合はフルパスで
.\tough.bat examples\hello.tough

# LLVM IR（中間コード）を表示
python main.py --emit-ir examples\hello.tough
```

## 文字化けする場合

Windows のコマンドプロンプトや PowerShell では、文字コードが Shift-JIS になっていると文字化けします。
`tough.bat` 経由で実行すれば自動的に UTF-8 (65001) に切り替わりますが、手動で python コマンドを使う場合は以下を実行してください。

```cmd
chcp 65001
python main.py examples\hello.tough
```

## 構文一覧

| カテゴリ | TOUGH 構文 | 意味 |
|---|---|---|
| 開始 | `我が名は　尊鷹` | プログラム開始 |
| 終了 | `逃げるんかいっ` | プログラム終了 |
| 変数宣言 | `Xだ Xが正体を現すぞ` | 変数の宣言 |
| 代入 | `(値) を継ぐ (変数)` | 変数への代入 |
| 出力 | `(値) しゃあっ` | 画面に出力 |
| 入力 | `(変数) を教えてくれよ` | キーボード入力 |
| 増加 | `(変数) 進化したと言うてくれや` | +1 |
| 減少 | `(変数) （哀）` | -1 |
| 等しい | `ガチンコ` | == |
| 等しくない | `ガチンコじゃない` | != |
| 大きい | `を超えた` | > |
| 小さい | `に及ばない` | < |
| if | `なにっ ((条件)) { ... }` | 条件分岐 |
| elif | `いやちょっとまてよ ((条件)) { ... }` | 追加条件 |
| else | `う　あ　あ　あ　あ（ＰＣ書き文字） { ... }` | それ以外 |
| while | `禁断の"((条件))度打ち" { ... }` | ループ |
| 関数 | `自分たちの手で作るから尊いんだ ...` | 関数定義 |
| 例外送出 | `はっきり言ってそれって病気だから お前死ぬよ` | 例外を投げる |
| コメント | `（○○のコメント）コメント内容` | コメント |

## サンプル: Hello World

```
（祠部矢のコメント）Hello World プログラム

我が名は　尊鷹

「こんにちは、世界！」 しゃあっ
「TOUGHへようこそ！」 しゃあっ

逃げるんかいっ
```

## サンプル: FizzBuzz

```
（祠部矢のコメント）FizzBuzz プログラム

我が名は　尊鷹

iだ iが正体を現すぞ
1 を継ぐ i

禁断の"i に及ばない 31 度打ち" {
    なにっ (i % 15 ガチンコ 0) {
        「FizzBuzz」 しゃあっ
    }
    いやちょっとまてよ (i % 3 ガチンコ 0) {
        「Fizz」 しゃあっ
    }
    いやちょっとまてよ (i % 5 ガチンコ 0) {
        「Buzz」 しゃあっ
    }
    う　あ　あ　あ　あ（ＰＣ書き文字） {
        i しゃあっ
    }
    i 進化したと言うてくれや
}

逃げるんかいっ
```
